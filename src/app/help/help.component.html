How to use Boneify:

  1. Population - Choose the population of bones whose data you will be comparing your results to.
  2. Sex - Select the biological sex of your comparison subject. Biological sex determines which set of statistics must be used during calculations.
  3. Bone - Choose what type of bone you are comparing. Clavicle is only available in the US population set
  4. ROC Graph - Displays the progression of the three different formats of the equation: using only mean, using only standard deviation, or using both. Using this graph, choose which equation is most effective for you in the Select Calculation Choice field
  5. Calculation Choice - Select what statistical measurements you will be entering into the equation: mean, standard deviation, or both.
